## 실습 1-3. 프롬프트 체이닝 — 대화로 답 다듬기 (15분) ⭐

> **핵심 기술.** 한 번에 완벽한 답을 기대하지 말고, **이어서 요청하며 좁혀간다.**

### STEP 1~4. 아래 순서대로 하나씩 이어 입력
🖥️ ChatGPT에서 **같은 대화창 안에서** 순서대로:
1. 💬 `지역별 매출을 요약해줘` → 일반적인 답이 온다

3. 💬 `각 지역 1위 제품도 열로 추가해줘` → 열이 추가된다

✅ **체크포인트:** 4번의 짧은 요청으로 "요약 → 표 → 확장 → 보고문"까지 완성됐다.
🤖 **AI 팁:** 답이 아쉬우면 `더 짧게` `이유도 붙여서` `표로`처럼 **이어서** 요청하라. 이것이 AI 활용의 가장 큰 힘이다.

## 실습 1-4. 물려받은 수식 거꾸로 해독 (15분)

> **상황:** 전임자가 만든 복잡한 수식이 뭘 하는지 모른다.


# 2교시 — 함수 자동화: 집계·조회·병합·계산 (60분)

**이 교시가 끝나면:** 핵심 함수를 AI로 만들고, 두 표 병합·급여 계산 같은 실무 계산까지 해낸다.
📄 2-1~2-4, 2-7: `매출데이터` 시트 / 2-5: `주문내역`·`제품마스터` 시트 / 2-6: `근태` 시트

## 실습 2-1. 조건부 합계 SUMIFS (8분)

### STEP 1. 파일과 셀 준비
🖥️ `AI엑셀실습_통합.xlsx`의 **매출데이터** 시트를 열고 빈 셀 **J1**을 클릭한다.

### STEP 2. 맥락을 담아 AI에 요청
💬
```
너는 엑셀 전문가야. G=금액, C=부서, H=월인 표(2행부터 200행)가 있어.
'영업1팀'의 '1월' 금액 합계를 구하는 SUMIFS 함수를 만들어줘.
```

### STEP 3. 받은 함수를 J1에 붙여넣고 Enter
```excel
=SUMIFS(G:G, C:C, "영업1팀", H:H, 1)
```
✅ **체크포인트:** J1 = **110,020,000**

### STEP 4. 조건을 바꿔 확인
🖥️ `"영업1팀"`을 `"영업2팀"`, `"영업3팀"`으로 바꿔본다.
✅ 영업2팀 **37,400,000** / 영업3팀 **21,070,000**

⚠️ 텍스트 조건의 따옴표(`" "`)를 빼면 오류. 범위를 일부만 잡아도 값이 틀린다.
🤖 **AI 팁:** 받은 뒤 `이 함수의 각 부분이 뭘 하는지 한 줄씩 설명해줘`라고 물어 **이해하고 넘어가는 습관**을 들이자.

## 실습 2-2. 개수·평균 COUNTIFS·AVERAGEIFS (7분)

### STEP 1. 두 함수를 한 번에 요청
💬
```
같은 표에서 1) 서울에서 팔린 노트북 건수(COUNTIFS)
2) 노트북의 평균 금액(AVERAGEIFS)을 각각 만들어줘. B=지역, D=제품, G=금액.
```

### STEP 2. J2·J3에 각각 입력
```excel
=COUNTIFS(B:B, "서울", D:D, "노트북")     ' J2
=AVERAGEIFS(G:G, D:D, "노트북")           ' J3
```
✅ **체크포인트:** J2 = **6건** / J3 = **11,400,000**

## 실습 2-3. 값 조회 3형제 — XLOOKUP·INDEX-MATCH·VLOOKUP (10분)

### STEP 1. 조회 대상 입력
🖥️ **J5** 셀에 `태블릿` 입력.

### STEP 2. 세 방식 비교 요청
💬 `J5의 제품명으로 단가(F열)를 찾는 함수를 XLOOKUP / INDEX-MATCH / VLOOKUP 세 방식으로 만들고 차이를 알려줘. 제품은 D열, 단가는 F열.`

### STEP 3. K5·K6·K7에 각각 입력
```excel
=XLOOKUP(J5, D:D, F:F)              ' 최신·권장 (365)
=INDEX(F:F, MATCH(J5, D:D, 0))     ' 모든 버전 OK, 왼쪽 조회 가능
=VLOOKUP(J5, D:F, 3, 0)            ' 가장 익숙, 찾는 값이 맨 왼쪽일 때만
```
✅ **체크포인트:** 세 함수 모두 **800,000** (결과 동일, 호환성만 다름)
💡 회사 PC가 구버전이면 INDEX-MATCH를 쓰면 된다.

## 실습 2-4. 동적 배열 FILTER·SORT (8분) ⭐

### STEP 1. 새 시트에 요청 (엑셀 365)
💬 `엑셀 365를 써. A2:H201에서 G열(금액)이 1,000만원 이상인 행만 뽑아 금액 내림차순 정렬하는 함수를 만들어줘.`

### STEP 2. 빈 셀에 입력
```excel
=SORT(FILTER(매출데이터!A2:H201, 매출데이터!G2:G201>=10000000), 7, -1)
```
> 읽는 법: FILTER = 조건에 맞는 행만 / SORT의 `7` = 7번째 열(금액) 기준, `-1` = 내림차순.

✅ **체크포인트:** **38건**이 자동으로 펼쳐진다(spill).
⚠️ 아래 칸에 다른 값이 있으면 `#SPILL!` → 그 칸을 비우면 해결.
💡 원본이 바뀌면 **자동 갱신** → "매달 갱신되는 조건별 목록"에 최적.

## 실습 2-5. 두 표 병합 — 주문내역 + 제품마스터 (12분) ⭐ [현업 실습 A]

📄 같은 파일의 `주문내역`·`제품마스터` 시트
> **상황:** 주문내역엔 제품**코드만** 있고, 제품명·단가는 마스터에 따로 있다. **실무 최다 기술.**

### STEP 1. 시트 확인
🖥️ `주문내역` 시트를 열면 `주문번호·제품코드·수량`만 있다(D열부터 비어 있음).

### STEP 2. AI에 요청
💬
```
주문내역 시트의 B열 제품코드로 제품마스터 시트에서 제품명과 단가를
가져오는 XLOOKUP을 만들어줘. 마스터는 A열 코드, B열 제품명, C열 단가야.
```

### STEP 3. D2·E2에 입력하고 6행까지 채우기
```excel
=XLOOKUP(B2, 제품마스터!A:A, 제품마스터!B:B)   ' D2: 제품명
=XLOOKUP(B2, 제품마스터!A:A, 제품마스터!C:C)   ' E2: 단가
```

### STEP 4. 금액과 총액
🖥️ **F2** `=C2*E2` 입력 후 채우기 → **F7** `=SUM(F2:F6)`

✅ **체크포인트:**
| 주문 | 제품명 | 단가 | 금액 |
|------|--------|------|------|
| 1001 | 노트북 | 1,200,000 | 2,400,000 |
| 1002 | 태블릿 | 800,000 | 800,000 |
| 1003 | 키보드 | 50,000 | 500,000 |
| 1004 | 모니터 | 300,000 | 900,000 |
| 1005 | 노트북 | 1,200,000 | 1,200,000 |
→ 주문 **총액 5,800,000원**

### STEP 5. 제품별 수량 집계 & 최고가 제품
```excel
=SUMIF(주문내역!B:B, "NB", 주문내역!C:C)                        ' 노트북 총수량
=XLOOKUP(MAX(제품마스터!C:C), 제품마스터!C:C, 제품마스터!B:B)     ' 최고가 제품명
```
✅ 노트북 **3개** / 키보드 10 / 모니터 3 / 태블릿 1 · 최고가 = **노트북**

⚠️ 코드가 마스터에 없으면 `#N/A` → `=IFERROR(XLOOKUP(...), "코드확인")`으로 감싸면 보고서가 안 깨진다.
💡 사번→부서·직급, 거래처코드→상호 등 "코드로 이름 붙이기"는 전부 같은 원리.

## 실습 2-6. 근무시간·급여 계산 — 근태 (10분) [현업 실습 B]

📄 같은 파일의 `근태` 시트
> **규칙:** 09:00 출근, 점심 1시간 제외, 18시 이후는 1.5배 수당, 시급 15,000원.

### STEP 1. 근무시간 (E2)
💬 `출근(B)·퇴근(C) 시각으로 근무시간을 구해줘. 점심 1시간 빼고. 시간이 텍스트로 저장돼 있을 수도 있어.`
```excel
=(TIMEVALUE(C2)-TIMEVALUE(B2))*24 - 1
```
> 💡 시간 형식이라면 `=(C2-B2)*24-1`로도 됨. 텍스트라면 위처럼 TIMEVALUE로 감싼다.

### STEP 2. 야근시간 (F2) — 18시 이후만
```excel
=MAX(0, (TIMEVALUE(C2)-TIME(18,0,0))*24)
```
> 읽는 법: 퇴근−18시가 음수(야근 없음)면 MAX가 0으로 만들어 준다.

### STEP 3. 총 급여 (G2) — 정규(최대 8h) + 야근×1.5
```excel
=MIN(E2,8)*D2 + F2*D2*1.5
```

### STEP 4. 야근 여부 (H2)
```excel
=IF(F2>0, "야근", "정상")
```

✅ **체크포인트:**
| 사원 | 퇴근 | 근무 | 야근 | 급여 | 판정 |
|------|------|------|------|------|------|
| 김영업 | 18:00 | 8h | 0h | 120,000 | 정상 |
| 이기획 | 20:00 | 10h | 2h | 165,000 | 야근 |
| 박개발 | 22:00 | 12h | 4h | **210,000** | 야근 |
| 최디자인 | 19:00 | 9h | 1h | 142,500 | 야근 |
| 정인사 | 18:30 | 8.5h | 0.5h | 131,250 | 야근 |

💡 시간제 강사료·아르바이트 급여·초과근무 수당 계산에 그대로. 시급·배율만 바꾸면 된다.

## 실습 2-7. 차등 수수료(중첩 IFS) + 누계 (5분)

📄 `매출데이터` 시트
### STEP 1. 구간별 수수료
💬 `G열 금액에 대해 500만원 이상 5%, 100만원 이상 3%, 그 미만 1% 수수료를 계산하는 함수를 만들어줘.`
```excel
=G2 * IFS(G2>=5000000, 0.05, G2>=1000000, 0.03, TRUE, 0.01)
```
✅ 6,000,000 → **300,000** / 1,200,000 → **36,000** / 300,000 → **3,000**

### STEP 2. 누계(러닝토탈)

```excel

```
✅ 월별 누계: 1월 168,490,000 → 6월 **978,280,000**

📄 같은 파일의 `고객RFM` 시트
> RFM = 최근성(**R**ecency)·빈도(**F**requency)·금액(**M**onetary)으로 우수 고객을 가려내는 대표 기법.

### STEP 1. 최근성 = 며칠 전 구매했나 (E2)
```excel

```
✅ A상사 5일 · B물산 60일 · C유통 20일 · D전자 76일 · E마트 2일

### STEP 2. R·F·M 점수 (F·G·H열)
💬
```
고객별 점수 함수를 만들어줘.
- R: 15일 이내 3점, 30일 이내 2점, 그 외 1점
- F(구매횟수): 10회 이상 3점, 5회 이상 2점, 그 외 1점
- M(누적금액): 200만 이상 3점, 100만 이상 2점, 그 외 1점
```
```excel
=IFS(E2<=15,3, E2<=30,2, TRUE,1)                 ' F열: R
=IFS(C2>=10,3, C2>=5,2, TRUE,1)                  ' G열: F (C=구매횟수)
=IFS(D2>=2000000,3, D2>=1000000,2, TRUE,1)       ' H열: M (D=누적금액)
```

### STEP 3. 총점(I2)·등급(J2)·VIP만 뽑기
```excel
=F2+G2+H2
=IFS(I2>=8,"VIP", I2>=6,"우수", TRUE,"일반")
=FILTER(A2:A6, I2:I6>=8)      ' 엑셀 365 — VIP만 추출
```

✅ **체크포인트:**
| 고객 | R | F | M | 총점 | 등급 |
|------|---|---|---|------|------|
| A상사 | 3 | 3 | 3 | 9 | **VIP** |
| E마트 | 3 | 3 | 3 | 9 | **VIP** |
| C유통 | 2 | 2 | 2 | 6 | 우수 |
| B물산 | 1 | 1 | 1 | 3 | 일반 |
| D전자 | 1 | 1 | 1 | 3 | 일반 |

💡 VIP엔 재구매 혜택, 이탈위험(낮은 R: D전자·B물산)엔 재방문 안내 → **마케팅 예산을 데이터로 배분**.

🖥️ ChatGPT에 `AI엑셀실습_통합.xlsx` 업로드. (`정제실습` 시트에 공백·날짜형식·₩기호·빈 행·중복이 섞여 있음)

```
업로드한 파일의 '정제실습' 시트를 정제해줘.
1) 텍스트 앞뒤 공백 제거
2) 날짜를 YYYY-MM-DD로 통일 (yy.m.d 형식은 2024년으로 해석)
3) 금액을 숫자로 변환(₩, 콤마 제거)
```

✅ **체크포인트:** **8행 → 5행**

### 🔴 STEP 3. 검증의 함정 직접 확인 (이 과정 전체에서 가장 중요)
🖥️ 정제 결과에서 `24.1.7`이 들어 있던 행의 날짜를 확인한다.
> `24.1.7`을 자동 변환 도구가 **2007-01-24로 잘못 인식**하는 사고가 실제로 난다.

> - 올바른 결과 ✅: **2024-01-07**
>
> **대응 2가지:** ① 프롬프트에 "yy.m.d는 2024년으로 해석" **명시** ② 결과를 **눈으로 검증**.
> 공문·보고서에 이런 오류가 들어가면 되돌릴 수 없다. **"AI는 검증해야 한다"의 살아있는 증거.**

## 실습 5-2. 텍스트 분할·추출 + 정규식 (13분) ⭐

### STEP 1. 주소에서 시·구 분리 (A2 = `서울시 강남구 테헤란로 123`)
```excel
=TEXTBEFORE(A2, " ")                        ' 시 → 서울시
=TEXTBEFORE(TEXTAFTER(A2, " "), " ")        ' 구 → 강남구
```
✅ **서울시 / 강남구**

### STEP 2. 코드에서 분류·연도 추출 (D2 = `NB-2024-001`)
```excel
=LEFT(D2, FIND("-",D2)-1)             ' NB
=MID(D2, FIND("-",D2)+1, 4)           ' 2024
```
✅ **NB / 2024**
⚠️ TEXTBEFORE/AFTER는 365 신함수 → 구버전은 "LEFT·MID·FIND 조합으로 바꿔줘"라고 AI에 요청.

### STEP 3. 정규식으로 이메일·전화 뽑기
💬
```
"문의: hong@company.com / 010-1234-5678" 같은 텍스트에서
이메일과 전화번호만 각각 뽑는 방법을 알려줘. 정규식 또는 엑셀 함수로.
```
✅ **체크포인트:** 이메일 `hong@company.com`, 전화 `010-1234-5678`
💡 Sheets는 `=REGEXEXTRACT(A2, "[\w.]+@[\w.]+")`로 바로 가능.

## 실습 6-2. 여러 시트 통합 매크로 (12분)

### STEP 1. AI에 요청 (⚠️ 대상 시트를 명시!)
> 이 파일엔 매출데이터·근태 등 **다른 시트도 함께** 있다. "모든 시트"가 아니라 **'1월','2월','3월'만** 합치라고 명시해야 한다.
💬
```
이 파일에는 여러 시트가 있어. 그중 '1월','2월','3월' 시트의 데이터만
'통합' 시트로 합쳐줘. 각 시트 1행은 헤더이고, 통합엔 헤더를 한 번만.
다른 시트는 건드리지 마. 초보자용 주석도 달아줘.
```

### STEP 2. 받은 코드를 모듈에 붙여넣기
```vba
Sub 시트통합()
    Dim names As Variant, i As Long
    Dim ws As Worksheet, target As Worksheet
    Dim lastRow As Long, pasteRow As Long

    names = Array("1월", "2월", "3월")        ' ★ 합칠 시트만 지정

    Application.DisplayAlerts = False
    On Error Resume Next
    Worksheets("통합").Delete                ' 기존 통합 시트 있으면 삭제
    On Error GoTo 0
    Application.DisplayAlerts = True
    Set target = Worksheets.Add
    target.Name = "통합"

    pasteRow = 1
    For i = LBound(names) To UBound(names)
        Set ws = Worksheets(names(i))
        lastRow = ws.Cells(ws.Rows.Count, 1).End(xlUp).Row
        If i = 0 Then                        ' 첫 시트: 헤더 포함
            ws.Range("A1:Z" & lastRow).Copy target.Cells(pasteRow, 1)
            pasteRow = pasteRow + lastRow
        Else                                 ' 이후: 헤더 빼고(2행부터)
            If lastRow >= 2 Then
                ws.Range("A2:Z" & lastRow).Copy target.Cells(pasteRow, 1)
                pasteRow = pasteRow + (lastRow - 1)
            End If
        End If
    Next i
    Application.CutCopyMode = False
    MsgBox "통합 완료! 총 " & (pasteRow - 1) & "행", vbInformation
End Sub
```

### STEP 3. 실행
🖥️ 개발 도구 → 매크로 → `시트통합` → **실행**.
✅ **체크포인트:** "통합 완료! 총 **21행**" (1월 9 + 2월 6 + 3월 6)
🤖 **AI 팁:** 실행 전 `이 코드가 무슨 일을 하는지 줄별로 설명해줘`로 학습. VBA를 몰라도 "무엇을 자동화할지 한글로 설명"만 하면 코드가 나온다.

## 실습 6-3. 대화형 매크로 — 물어보고 추출하기 (10분) ⭐신규

> **재미 포인트:** 매크로가 **나에게 질문**하고, 대답에 따라 다르게 움직인다. (InputBox 활용)

### STEP 1. AI에 요청
💬
```
'매출데이터' 시트(A~H열, 1행 헤더)에서 매크로를 실행하면
InputBox로 제품명을 물어보고, 입력한 제품의 행만
새 시트로 복사해주는 VBA를 만들어줘. 몇 행을 추출했는지도 알려줘.
```

### STEP 2. 붙여넣고 실행
```vba
Sub 제품추출()
    Dim prod As String, src As Worksheet, dst As Worksheet
    Dim lastRow As Long, r As Long, pasteRow As Long

    prod = InputBox("어떤 제품을 추출할까요?" & vbLf & "(노트북/모니터/키보드/마우스/태블릿)", "제품 추출기")
    If prod = "" Then Exit Sub               ' 취소하면 종료

    Set src = Worksheets("매출데이터")
    Application.DisplayAlerts = False
    On Error Resume Next
    Worksheets(prod & "_추출").Delete
    On Error GoTo 0
    Application.DisplayAlerts = True
    Set dst = Worksheets.Add
    dst.Name = prod & "_추출"

    src.Rows(1).Copy dst.Rows(1)             ' 헤더 복사
    pasteRow = 2
    lastRow = src.Cells(src.Rows.Count, 1).End(xlUp).Row
    For r = 2 To lastRow
        If src.Cells(r, 4).Value = prod Then ' D열 = 제품
            src.Rows(r).Copy dst.Rows(pasteRow)
            pasteRow = pasteRow + 1
        End If
    Next r
    MsgBox prod & " " & (pasteRow - 2) & "행 추출 완료!", vbInformation
End Sub
```

### STEP 3. "노트북"으로 테스트
🖥️ 실행 → 입력창에 `노트북` → 확인.
✅ **체크포인트:** "노트북 **42행** 추출 완료!" (태블릿=49 / 모니터=38 / 키보드=37 / 마우스=34)

💡 **실무 응용:** "어느 팀? 어느 달?"을 물어보는 추출기로 확장하면, 누가 써도 실수 없는 **팀 공용 도구**가 된다.

## 실습 6-4. 시트 자동 분리 — 통합의 반대 (10분) ⭐신규

> **재미 포인트:** 버튼 한 번에 매출데이터가 **지역별 5개 시트로 착착 분리**된다. 배포용 파일 만들 때의 단골 작업.

### STEP 1. AI에 요청
💬
```
'매출데이터' 시트를 B열(지역) 값별로 나눠서,
지역 이름의 시트를 각각 만들고 해당 행만 복사하는 VBA를 만들어줘.
각 시트 1행엔 헤더를 넣어줘.
```

### STEP 2. 붙여넣고 실행
```vba
Sub 지역별분리()
    Dim src As Worksheet, dst As Worksheet
    Dim lastRow As Long, r As Long
    Dim region As String

    Set src = Worksheets("매출데이터")
    lastRow = src.Cells(src.Rows.Count, 1).End(xlUp).Row

    For r = 2 To lastRow
        region = src.Cells(r, 2).Value        ' B열 = 지역
        On Error Resume Next
        Set dst = Worksheets(region)
        On Error GoTo 0
        If dst Is Nothing Then                 ' 그 지역 시트가 없으면 새로 만들기
            Set dst = Worksheets.Add(After:=Worksheets(Worksheets.Count))
            dst.Name = region
            src.Rows(1).Copy dst.Rows(1)       ' 헤더
        End If
        src.Rows(r).Copy dst.Cells(dst.Rows.Count, 1).End(xlUp).Offset(1, 0)
        Set dst = Nothing
    Next r
    MsgBox "지역별 시트 분리 완료!", vbInformation
End Sub
```

### STEP 3. 결과 확인
✅ **체크포인트:** 서울·부산·대구·인천·광주 시트가 생기고, 데이터 행 수가
**서울 44 / 인천 46 / 대구 40 / 부산 39 / 광주 31** (합계 200)이면 성공.

⚠️ 다시 실행하면 중복으로 쌓인다 → AI에 `실행할 때 기존 지역 시트를 먼저 지우게 수정해줘`라고 개선 요청 (개선 요청도 연습!).

## 실습 6-5. 원클릭 요약 대시보드 + 실행 버튼 (10분) ⭐신규

> **재미 포인트:** 시트 위의 **버튼을 클릭**하면 부서별 요약표가 그 자리에서 만들어진다 — "매크로를 도구로 포장"하는 법.

### STEP 1. 요약 매크로 요청
💬
```
'요약' 시트를 만들어 부서별(영업1~3팀) 총매출을 표로 자동 작성하는
VBA를 만들어줘. '매출데이터' 시트의 C열=부서, G열=금액이야.
제목과 작성일시도 넣고, 금액엔 천단위 콤마 서식.
```
```vba
Sub 요약대시보드()
    Dim ws As Worksheet, i As Long
    Dim depts As Variant
    depts = Array("영업1팀", "영업2팀", "영업3팀")

    Application.DisplayAlerts = False
    On Error Resume Next
    Worksheets("요약").Delete
    On Error GoTo 0
    Application.DisplayAlerts = True
    Set ws = Worksheets.Add(Before:=Worksheets(1))
    ws.Name = "요약"

    ws.Range("A1").Value = "부서별 매출 요약"
    ws.Range("A1").Font.Size = 16: ws.Range("A1").Font.Bold = True
    ws.Range("A2").Value = "작성: " & Format(Now, "yyyy-mm-dd hh:mm")
    ws.Range("A4").Value = "부서": ws.Range("B4").Value = "총매출"
    ws.Range("A4:B4").Font.Bold = True

    For i = 0 To 2
        ws.Cells(5 + i, 1).Value = depts(i)
        ws.Cells(5 + i, 2).Value = Application.WorksheetFunction.SumIf( _
            Worksheets("매출데이터").Range("C:C"), depts(i), _
            Worksheets("매출데이터").Range("G:G"))
    Next i
    ws.Range("B5:B7").NumberFormat = "#,##0"
    ws.Columns("A:B").AutoFit
    MsgBox "요약 완성!", vbInformation
End Sub
```
✅ **체크포인트:** 영업1팀 **501,940,000** / 영업2팀 **243,090,000** / 영업3팀 **233,250,000**

### STEP 2. 버튼 달기
🖥️ 개발 도구 → **삽입 → 단추(양식 컨트롤)** → 시트에 드래그로 그리기 → 매크로 지정 창에서 `요약대시보드` 선택 → 버튼 이름을 "요약 새로고침"으로 수정.
✅ **체크포인트:** 버튼 클릭 → 요약이 다시 만들어지고 작성일시가 갱신된다.
💡 **이 순간이 자동화의 완성** — 매크로를 모르는 동료도 버튼만 누르면 된다.

## 실습 6-6. 디버깅 + 느린 파일 진단 (10분)

### STEP 1. 일부러 오류 내고 AI에 수리 맡기기
🖥️ 코드의 `Worksheet`를 `Worksheeet`로 바꿔 실행 → 오류 발생.
💬 `이 VBA에서 이런 오류가 나: [오류 메시지 붙여넣기]. 원인과 수정 코드를 알려줘.`
✅ AI가 오타를 짚어 수정해 준다 → **AI = 디버깅 파트너**.

### STEP 2. 느린 파일 진단
💬 `엑셀 파일이 너무 느려. 흔한 원인과 점검·해결 방법을 순서대로 알려줘.`
✅ AI 답 요지: ① 전체 열 참조·휘발성 함수(NOW·INDIRECT) 과다 ② 조건부 서식 누적 ③ 안 쓰는 서식·이미지 ④ 수동 계산 전환으로 원인 격리.

# 7교시 — 종합 미니프로젝트: ChatGPT 하나로 끝내는 월간 경영 보고서 (60분)

**이 교시가 끝나면:** ChatGPT **한 대화창** 안에서 진단→정제→분석→시각화→예측→자동화→보고까지 **7단계**를 연결해 완성한다.
📄 `AI엑셀실습_통합.xlsx` (매출데이터·정제실습 시트) · 도구: **ChatGPT만 사용**

> **프로젝트 규칙**
> - 모든 단계를 **하나의 ChatGPT 대화창**에서 진행한다 (1교시의 프롬프트 체이닝 실전!)
> - 각 단계 끝의 ✅를 확인하고 다음 단계로 넘어간다
> - 막히면 "다시", "더 자세히", "표로"처럼 **이어서** 요청한다

## 미니프로젝트 7단계 (45분)

### STEP 1. 데이터 진단 — 분석 전에 상태부터 (5분)
🖥️ ChatGPT에 `AI엑셀실습_통합.xlsx` 업로드 후:
💬
```
이 파일의 '매출데이터'와 '정제실습' 시트를 각각 진단해줘.
행 수, 열 구성, 결측치·이상치·형식 오류 여부를 표로 정리하고,
분석 전에 손봐야 할 점을 알려줘.
```
✅ **체크포인트:** 매출데이터 200행 = "깨끗", 정제실습 = 공백·날짜형식·₩기호·중복 문제 지적.

### STEP 2. 정제 — 규칙을 명시하고 검증 (5분)
💬
```
'정제실습' 시트를 정제해줘.
1) 공백 제거 2) 날짜 YYYY-MM-DD 통일 (yy.m.d는 2024년으로 해석)
3) 금액 숫자화(₩·콤마 제거) 4) 빈 행·완전 중복 제거.
정제 전후 행 수와 정제된 표를 보여줘.
```
✅ **체크포인트:** 8행 → **5행** · 날짜에 **2024-01-07** 확인 (2007이면 재요청!)
> 🔴 5교시의 날짜 함정을 실전에서 스스로 검증하는 단계다.

### STEP 3. 분석 — 피벗·구성비·월별 추세 (8분)
💬
```
이제 '매출데이터' 시트를 분석해줘.
1) 부서×월 매출 피벗 2) 제품별 매출 구성비(%)
3) 월별 매출 추세와 전월 대비 증감률
핵심 발견 3가지를 한 줄씩 덧붙여줘.
```
✅ **체크포인트:** 총매출 **978,280,000** · 노트북+태블릿 **87%** · 5월 **+93.1%**, 4월 **−33.4%**

### STEP 4. 시각화 — 차트 2종 (7분)
💬
```
위 분석으로 차트 2개를 그려줘.
1) 월별 매출 추세 선 그래프 (증감이 큰 달 표시)
2) 제품별 구성비 막대 그래프
각 차트 아래 한 줄 해설도.
```
✅ **체크포인트:** 5월 봉우리·4월 골짜기가 보이는 추세선 + 노트북·태블릿이 압도적인 막대.

### STEP 5. 예측 — 근거를 요구하라 (7분)
💬
```
월별 추세를 바탕으로 7월 매출을 예측해줘.
단순평균과 추세선 두 방식으로 각각 구하고,
어느 쪽을 왜 더 신뢰할지, 5월 급등이 예측을 어떻게 왜곡할 수 있는지 설명해줘.
```
✅ **체크포인트:** 두 방식의 값이 **다르게** 나오고, AI가 "이상치(5월) 때문에 오차 가능"을 인정하면 성공.
⚠️ 예측치는 보고서에 "참고 전망"으로만 표기.

### STEP 6. 자동화 — 요약 매크로를 받아 심는다 (7분)
💬
```
이 보고서를 매달 재사용할 수 있게, 엑셀에서 '월간요약' 시트를 자동 생성하는
VBA를 만들어줘. 내용: 제목·작성일·부서별 총매출(천단위 콤마)·월별 매출 표.
'매출데이터' 시트의 C열=부서, G열=금액, H열=월이야.
```
🖥️ 받은 코드를 통합 파일(.xlsm)의 모듈에 붙여넣고 실행.
✅ **체크포인트:** 부서별 합계가 영업1팀 **501,940,000** / 영업2팀 **243,090,000** / 영업3팀 **233,250,000**이면 성공.
> 6교시에서 배운 것을 ChatGPT 대화 안에서 그대로 재현하는 단계 — "코드도 대화의 산출물"이다.

### STEP 7. 보고 — 요약문 + 메일 초안 (6분)
💬
```
지금까지의 분석을 종합해서
1) 경영진 보고용 3줄 요약 (수치 포함)
2) 팀장에게 보낼 보고 메일 초안 (제목 포함, 정중하고 간결하게)
을 작성해줘.
```
✅ **체크포인트:** 총매출 9.78억 · 5월 +93%/4월 −33% · 상위 2개 제품 87%가 요약과 메일에 정확히 들어가 있는지 **원본 수치와 대조**.
🖥️ 이어서 체이닝으로 다듬기: `더 간결하게` `개조식으로` `숫자를 앞으로`.

## 프로젝트 회고 (7분)
